# MemDDC-LongMemory-Compression-DDD
Trae Skill：长记忆持久化+双层智能压缩+DDD文档契约驱动的AI编码重构专家，支持无文档项目逆向生成文档与DDD模型。

## 核心功能
- 🧠 **长记忆持久化**：跨会话不遗忘架构决策、业务规则、重构取舍。
- 📦 **双层智能压缩**：文档轻量化 + 记忆结构化，适配LLM上下文限制。
- 📄 **DDD契约驱动**：严格按领域模型与文档规范编码，杜绝自由发挥。
- 🔧 **无文档逆向生成**：自动扫描代码生成全套文档与DDD领域模型。
- 🔁 **双向同步**：代码变更自动同步更新文档、模型、记忆库。
- 🏗️ **重构优化**：遗留系统解耦、架构治理、业务逻辑无损迁移。

## 解决痛点
- 传统AI编码：遗忘架构、上下文超限、破坏领域边界。
- 遗留系统：无文档、无模型、隐性业务多、重构风险高。
- 工程治理：代码与文档脱节、架构腐化、迭代混乱。

## 适用场景
- Java微服务/企业级复杂工程。
- 遗留系统大规模重构。
- 无文档/无模型项目逆向工程。
- 长期迭代架构治理。
- AI Agent标准化编码约束。

## Trae使用
1. 下载`MemDDC-LongMemory-Compression-DDD`文件夹。
2. 导入Trae：设置 → 规则和技能 → 技能 → 导入文件。
3. 触发使用：对话中提及`MemDDC`、`DDD重构`等关键词，或手动指定`使用MemDDC-LongMemory-Compression-DDD技能`。
4. 无文档项目：自动执行逆向生成，再开始编码/重构。
