# 俄罗斯方块网页小游戏

一个使用原生 HTML、CSS 和 JavaScript 实现的经典俄罗斯方块游戏。

![游戏截图](screenshot.png)

## 在线试玩

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 功能特性

### 核心游戏机制
- **7种经典方块**：I、O、T、S、Z、J、L 形状方块
- **方块控制**：移动、旋转、加速下落、直接落下
- **碰撞检测**：边界检测和方块堆叠检测
- **行消除**：完整行自动消除并计分
- **墙踢功能**：旋转时自动调整位置避免卡住

### 游戏系统
- **分数系统**：
  - 消除1行：100分 × 当前等级
  - 消除2行：300分 × 当前等级
  - 消除3行：500分 × 当前等级
  - 消除4行：800分 × 当前等级
  - 加速下落：+1分/格
  - 直接落下：+2分/格
- **等级系统**：每消除10行升一级，下落速度逐渐加快
- **预览功能**：显示下一个即将出现的方块

### 操作说明
| 按键 | 功能 |
|------|------|
| ← (左方向键) | 向左移动 |
| → (右方向键) | 向右移动 |
| ↓ (下方向键) | 加速下落 |
| ↑ (上方向键) | 旋转方块 |
| 空格键 | 直接落到底部 |
| P 键 | 暂停/继续游戏 |
| Enter 键 | 开始游戏 |

### 核心技术点

#### 1. HTML 结构 (index.html)
- 使用 Canvas 绘制游戏区域和下一个方块预览
- 分数面板显示当前分数、等级和消除行数
- 模态框显示游戏结束界面
- 响应式布局适配不同屏幕尺寸

#### 2. CSS 样式 (style.css)
- **渐变背景**：使用 CSS 渐变创建视觉效果
- **卡片式布局**：圆角、阴影营造现代感
- **动画效果**：
  - 按钮悬停动画
  - 游戏结束弹窗滑入动画
- **响应式设计**：媒体查询适配移动设备

#### 3. JavaScript 游戏逻辑 (tetris.js)

##### 游戏配置
```javascript
const COLS = 10;           // 游戏板列数
const ROWS = 20;           // 游戏板行数
```

使用二维数组定义7种方块形状，每种方块包含：
- `shape`: 形状矩阵（1表示有方块，0表示空）
- `color`: 方块颜色
